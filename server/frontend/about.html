AI Agent Competition Platform

CITS3200 (Professional Computing) Project, 2022 S2.

This project is aimed to provide a server and client interface which allows agents (written in Java or Python) on the client's machine to play games hosted on the server against other agents. The server provides functionalities for recording historical data about games, agents' performance and provide said data on demand through an API. The frontend you see now is built to consume the game data API and allow clients to view leaderboards and query data about games.

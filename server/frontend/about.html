About the Platform

This project aims to provide a server and client interface which allows Python & Java agents to compete against one another. The server provides the functionality of recording historical game data, agents' performance, and provides said data on demand through an API. The front-end interface is built to retrieve the game data from the API and allow the server user to view leaderboards and query further information regarding agents, bots and play statistics.
